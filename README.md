# DLE — GitHub Pages (Personnages)

## Contenu
- `index.html` : la page
- `characters.json` : les données (100 persos)
- `assets/` : JS + CSS
- `.nojekyll` : évite certains soucis GitHub Pages

## Mise en ligne
1. Upload tout ce dossier dans ton repo (à la racine)
2. Settings → Pages → Deploy from a branch → main / root
3. Ouvre l’URL GitHub Pages
