# OnePieceDLE — Classic (GitHub Pages)

## Upload (le plus simple)
1. Crée un repo GitHub (ex: `onepiece-dle`)
2. Upload **ces fichiers à la racine** :
   - `index.html`
   - `characters.json`
3. Settings → Pages → Branch: `main` / Folder: `/ (root)` → Save
4. Ouvre l’URL GitHub Pages (avec un `/` final)

## Notes
- Le jeu charge `characters.json` via une URL robuste compatible GitHub Pages/Safari.
- Les essais sont stockés par jour dans `localStorage`.
