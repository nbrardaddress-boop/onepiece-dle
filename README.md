# Site DLE — Personnages (One Piece)

⚠️ Fan-site / projet perso : personnages et noms issus de One Piece.

## Upload
1. Dézippe
2. Upload tout à la racine du repo (index.html, characters.json, assets/, .nojekyll)

## GitHub Pages
Settings → Pages → Deploy from a branch → main / (root)

## Modifier la liste
Édite `characters.json` puis commit, et clique “Recharger” sur le site.
