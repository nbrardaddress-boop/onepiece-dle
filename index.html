Choisis un mode

Chaque mode a un perso du jour (daily). Tu as des essais, des indices, et un historique local (dans ton navigateur).

Classic
Deviner un personnage (table d’indices).
Indices débloqués : Saga (5), Affiliation (10)
Fruit du démon
On te montre un fruit → devine le possesseur.
Indices : Type (4), Nom (8)
Citation
Une phrase → devine qui l’a dite.
Indices : Destinataire(s) (4), Saga (8)
Silhouette
Perso noir zoomé → chaque essai dézoome.
Indice : dézoom progressif
Vidéo
Extrait flouté → chaque essai défloute.
Indice : défloutage progressif

Base de données

Le site marche avec /data/characters.json (+ quotes.json, fruits.json, etc.). Dans ce dossier, je t’ai mis une base démo et des scripts pour générer une vraie base large (200+ persos) depuis l’API publique api.api-onepiece.com. (CORS est désactivé côté API, donc on génère le JSON localement.)

Tu peux déployer tel quel sur GitHub Pages, puis compléter les fichiers data/ et assets/ à ton rythme.
